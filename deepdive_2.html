Question 2

Answer the following questions about a WMS GetCapabilities request

  • What are the required parameters, and what do they represent?

    • VERSION
    This is the version of the request

    SERVICE
    This is the type of service

    REQUEST
    This is the name of the request

  • What is returned in response to a WMS GetCapabilities request?

    • An xml file with metadata about what can be requested from the service

    Question 3

    Answer the following questions about a WMS GetMap request

  • What are the required parameters, and what do they represent?

    • VERSION
    This is the version of the request

    REQUEST
    This is the name of the request

    LAYERS
    This is the map layer(s) that you want to retrieve, essentially the information you want your map to display

    STYLES
    This is the style of the map

    SRS
    This is the spatial reference system used for the map you are requesting

    BBOX
    This is the minimum and maximum coordinates of the map to be displayed, in the SRS units

    WIDTH
    This is the width of the map in pixels

    HEIGHT
    This is the height of the map in pixels

    FORMAT
    This is the image format in which the map you are requesting is output

  • What is returned in response to a WMS GetMap request?

    • An image of a map

  • What is the significance of transparency in WMS requests?

    • Transparency allows the background of a map to be transparent instead of white or some other color, however only gifs support transparency.

  • You want a map image representing three layers of data in a single JPEG for a specified area of interest.

    • WFS GetFeature

  • You want to retrieve data representing a digital elevation model (a raster dataset) in the form of a GeoTIFF.
